charAt retorna a localização de um caracter.
String nome = "Ricardo";
if(nome.charAt(1) == ‘a’) {

toUpperCase passa para letras maiusculas ou minusculas.

System.out.println(nome.toUpperCase());
System.out.println(sobrenome.toLowerCase());
